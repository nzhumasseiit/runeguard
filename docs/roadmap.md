- CLI policy checks
- demo poisoned README

- `runeguard run`

- GitHub Actions/CI mode
- policy templates
- HTML/JSON audit reports

## v0.5 — Advanced Linux Backend

- eBPF audit
- optional eBPF LSM experiments
- AppArmor/SELinux integrations
